Vim Syntax

Posted on 2021-03-16 Edited on 2021-04-19 In Learning , Computer , Vim/NeoVim
Symbols count in article: 215 Reading time ≈ 1 mins.

写 NeoVim 插件过程中，对 Vim 语法高亮的一些总结。

优先级

  1. 多个 Match 或 Region 的匹配项在相同位置起始时，最后定义的优先。
  2. Keyword 优先级高于 Match 和 Region。
  3. 在更靠前的位置起始的匹配项优先。

其他

  • 大小写：:h syn-case
  • 折叠深度：:h syn-foldlevel
  • 语法检查：:h syn-spell
